## 设计亮点
- **双视图切换：** 卡片视图和列表视图平滑切换
- **筛选抽屉：** 从右侧滑入，背景蒙版
- **卡片悬浮：** 鼠标悬停时卡片上浮，显示操作按钮
- **智能标签：** 项目卡片显示智能标签，颜色区分

## 交互说明
1. **视图切换：** 点击"卡片视图"或"列表视图"按钮
2. **筛选抽屉：** 点击"筛选"按钮打开，点击蒙版或×关闭
3. **项目卡片：** 点击查看详情，悬停显示操作按钮
4. **批量操作：** 选择多个项目后显示批量操作工具栏
5. **分页：** 点击页码或上下页按钮切换

## 页面元素
- 导航栏（Logo、用户头像、项目列表、标签管理）
- 工具栏（视图切换、筛选、搜索、批量操作）
- 项目卡片/列表（9个，分页显示）
- 分页控件
- 筛选抽屉（标签筛选、语言筛选、时间筛选）

## 技术实现
- 视图切换使用 CSS Grid/Flexbox
- 筛选抽屉使用 React Portal + Framer Motion
- 虚拟滚动优化大量数据
- 搜索使用 debounce 优化
