## 设计亮点
- **AI摘要卡片：** 可展开收起，有平滑动画
- **README预览：** 代码高亮，图片懒加载
- **统计信息：** 数据可视化展示
- **相关推荐：** 智能推荐相关项目

## 交互说明
1. **AI摘要：** 点击"展开/收起"按钮
2. **README：** 滚动阅读，点击"查看更多"跳转到 GitHub
3. **标签管理：** 点击标签删除，点击"+添加标签"添加
4. **相关推荐：** 点击项目卡片跳转到对应详情页
5. **返回：** 点击"← 返回"回到项目列表页

## 页面元素
- 导航栏（Logo、返回按钮、项目名称、Star数）
- 项目基本信息（名称、描述、语言、Star、Fork）
- AI智能摘要卡片（可展开收起）
- README预览区域
- 统计信息卡片（提交数、Issue数、PR数）
- 标签管理区域
- 相关推荐项目卡片（3个）

## 技术实现
- README渲染使用 react-markdown
- 代码高亮使用 prism.js
- 数据统计使用 GitHub API
- 相关推荐使用协同过滤算法
