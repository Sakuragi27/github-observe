# 登录页原型图

## 页面布局
```
┌─────────────────────────────────────┐
│           GitHub Observe            │
│     智能管理你的 GitHub Stars        │
├─────────────────────────────────────┤
│  [账号登录]  [Token登录]              │
│                                     │
│  ┌─────────────────────────────┐   │
│  │  邮箱: [输入框]              │   │
│  │  密码: [输入框]              │   │
│  │  [登录按钮]                  │   │
│  └─────────────────────────────┘   │
│                                     │
│  或使用 GitHub OAuth 登录           │
│  [使用 GitHub 登录]                 │
│                                     │
│  还没有账号？[注册]                  │
└─────────────────────────────────────┘
```

## 设计亮点
- **极光渐变背景：** 蓝紫色渐变，动态流动效果
- **毛玻璃卡片：** 登录卡片有毛玻璃效果，背景模糊
- **智能引导：** 新用户有智能引导提示动画
- **响应式布局：** 适配移动端、平板、桌面

## 交互说明
1. **登录方式切换：** 点击"账号登录"或"Token登录"按钮切换
2. **表单验证：** 实时验证邮箱格式、密码长度、Token格式
3. **GitHub OAuth：** 点击跳转到 GitHub 授权页面
4. **智能引导：** 新用户显示引导动画，老用户直接登录

## 页面元素
- 标题：GitHub Observe
- 副标题：智能管理你的 GitHub Stars
- 登录方式切换按钮（2个）
- 邮箱输入框
- 密码输入框
- Token 输入框（Token登录时显示）
- 登录按钮
- GitHub OAuth 按钮
- 注册链接
- 智能引导提示（条件显示）

## 技术实现
- 使用 Next.js 的 Auth.js 处理认证
- GitHub OAuth 集成
- 表单验证使用 React Hook Form
- 动画使用 Framer Motion
